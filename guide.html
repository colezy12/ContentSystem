How to Use ContentSystem

A comprehensive guide to uploading and sharing files with ContentSystem

  1. Generate a GitHub Personal Access Token

üîë Step 1: Generate GitHub Token

To upload files, you need a GitHub Personal Access Token with repo scope:

  1. Go to GitHub Token Settings
  2. Name your token (e.g., "ContentSystem Uploader")
  3. Select the repo scope
  4. Click "Generate token"
  5. Copy the token (it starts with ghp_)
‚ö†Ô∏è Important: Keep your token secure! Don't share it with others. Your browser stores it locally, so you only need to enter it once.

üì§ Step 2: Upload Files

Using the Upload Interface

  1. Paste your GitHub token in the "GitHub Personal Access Token" field
  2. Click the upload box or drag and drop your files
  3. Select one or multiple files
  4. Click "Upload Files"
  5. Wait for the upload to complete

Supported File Types

  • Images: JPG, PNG, GIF, WebP, SVG, etc.
  • Videos: MP4, WebM, MOV, AVI, etc.
  • Audio: MP3, WAV, OGG, etc.
  • Documents: PDF, ZIP, TXT, and more

üîó Step 3: Share Links

After uploading, you'll get two types of links:

1. View Link

Format: https://colezy12.github.io/ContentSystem/view.html?id=XXXXXXXX

  • Shows a preview of the file
  • Includes file information
  • Has download buttons
  • Great for sharing with users

2. Direct Link

Format: https://raw.githubusercontent.com/.../files/XXXXXXXX.ext

  • Direct access to the file
  • Best for Discord embeds
  • Works for direct downloads
  • Can be used in HTML <img> and <video> tags

üí¨ Discord Integration

To embed your files in Discord:

  1. Copy the Direct Link (not the View Link)
  2. Paste it directly in your Discord message
  3. Discord will automatically embed images and videos
üí° Tip: The direct link must end with the file extension (e.g., .png, .mp4) for Discord to embed it properly.

üìä Recent Uploads

üîí Privacy & Security

‚ö†Ô∏è Important Privacy Information:
  • All uploaded files are stored in a public GitHub repository
  • Anyone with the link can access your files
  • Files are visible on the GitHub repository
  • Never upload sensitive or private content!
  • Your GitHub token is stored only in your browser's localStorage

‚ùì Troubleshooting

Upload Failed

  • Check that your GitHub token is valid
  • Ensure the token has repo scope
  • Try regenerating your token
  • Check your internet connection

File Not Loading

  • Verify the file ID in the URL is correct
  • Check that the file exists in the repository
  • Wait a few seconds for GitHub to process the upload

Discord Not Embedding

  • Use the Direct Link, not the View Link
  • Ensure the link ends with the file extension
  • Some file types may not be supported by Discord
  • Large video files may not embed (Discord has size limits)

üìù Best Practices

  • Use descriptive file names before uploading
  • Keep track of your upload links
  • Don't upload copyrighted content without permission
  • Be mindful of file sizes (GitHub has repository size limits)
  • Regularly clean up old files you no longer need
